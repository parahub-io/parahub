# Events

Community events and meetups with automatic chat rooms and map integration.

## Overview

Create and discover community events — from local meetups to online workshops. Every published event gets an automatic encrypted Matrix chat room where participants can coordinate.

## Event Types

- **Offline**: Physical meetups at a specific location (linked to a building on the map)
- **Online**: Virtual events with a video call link (Jitsi, Zoom, etc.)
- **Hybrid**: Both physical and online participation

## Creating an Event

1. Choose event type and category
2. Set title, description, date/time, and timezone
3. For offline/hybrid: pick a location on the map
4. For online/hybrid: add a video call URL
5. Optionally set a maximum number of participants
6. Upload a cover image
7. Publish — a Matrix chat room is automatically created

Events can be created by individual users or on behalf of an establishment.

## Participating

- **Join**: Register for an event and automatically join its Matrix chat room
- **Leave**: Unregister at any time
- **Status**: Mark yourself as Going or Maybe

## Discovery

Events are browsable with filters:
- Event type (offline, online, hybrid)
- Category
- City or country
- Date range
- Geographic proximity (nearby events)
- Text search

A dedicated "Events" tab in the directory page lists upcoming events. Past events remain browsable in a separate tab.

## Access Control

- **View events**: Public
- **Create events**: WoT 3+ (verified by 2 or more users)
- **Join events**: Any authenticated user
- **Edit/cancel events**: Organizer only
