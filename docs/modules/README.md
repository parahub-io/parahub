# Module Documentation

Detailed documentation for individual Parahub subsystems. See [Features](../features.md) for a high-level overview of all capabilities.

## Core

| Module | Description |
|--------|-------------|
| [Identity](identity.md) | Accounts, profiles, Web of Trust, PGP |
| [Marketplace](marketplace.md) | Items, pricing, barter, category matching |
| [Contracts](contracts.md) | P2P digital contracts, PGP signatures, arbitration |
| [Debts](debts.md) | P2P debt tracking, clearing via barter cycles |
| [Audit Trail](audit.md) | Cryptographic proofs, OpenTimestamps, court-ready export |
| [Communication](communication.md) | Matrix chat, Jitsi video calls, notifications |
| [Maps](maps.md) | MapLibre, geocoding, routing |
| [Directory](directory.md) | Business and organization directory, reviews, membership |
| [Blog & Mini-sites](cms.md) | Blog for users and organizations, custom mini-sites on subdomains, custom domains |
| [Video](video.md) | Self-hosted PeerTube video platform, upload, embed in posts, OIDC SSO |
| [Map Presence](map-presence.md) | Real-time MMORPG-style avatars on the map |
